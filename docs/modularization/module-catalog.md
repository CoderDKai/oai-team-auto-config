# 模块清单

## core

- 职责: 通用基础能力，包含配置、日志、通用工具函数
- 边界: 不直接包含业务流程，仅提供基础支撑
- 负责人: 待补充

## automation

- 职责: 浏览器自动化与流程编排
- 边界: 仅处理自动化流程，不直接包含业务服务逻辑
- 负责人: 待补充

## team

- 职责: Team 邀请、账号列表、Team 状态查询
- 边界: 仅处理 Team 相关 API 与流程
- 负责人: 待补充

## cpa

- 职责: CPA 系统授权与状态管理
- 边界: 仅包含 CPA 授权流程，不包含 CRS/S2A 逻辑
- 负责人: 待补充

## crs

- 职责: CRS 授权与账号入库
- 边界: 仅包含 CRS 相关 API 调用与入库逻辑
- 负责人: 待补充

## s2a

- 职责: S2A 授权与账号入库
- 边界: 仅包含 S2A 相关 API 调用与账号管理
- 负责人: 待补充

## email

- 职责: 邮箱创建、验证码获取、邮箱系统适配
- 边界: 仅处理邮箱能力，不包含 Team 或授权逻辑
- 负责人: 待补充

## cli

- 职责: 应用入口、命令参数、运行控制
- 边界: 不包含具体业务流程实现
- 负责人: 待补充
